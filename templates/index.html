Promotions API

Promotions API will help to manage promotions for sales on products. API supports several operations like viewing all current promotions, adding a promotion, deleting a promotion and many more.

Viewing all the available promotions

To list all active promotions, a GET request has to be made to the API

Adding a promotion

To create a promotion, a POST request has to be made the API

Deleting a promotion

To delete a promotion, a DELETE request has to be made at http://abc.com/promotions/delete/<promotion id>.

Updating a promotion

To update a promotion, a PUT request has to be made to the API

Cancelling a promotion

To cancel a promotion, a PUT request has to be made to the API

Query a promotion based on attribute id

To query a promotion based on attribute id, a GET request has to be made to the API and the parameters id need to be passed

Query a promotion based on attribute kind

To query a promotion based on attribute kind, a GET request has to be made to the API and the parameters kind need to be passed
